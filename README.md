# IGNIS v2 — Flight Computer

SRAD flight computer for Ignitia Rocket Lab — LASC 2026 (Brazil).  
4-layer PCB designed in Altium Designer.

## Key specs
- MCU: ESP32-S3-WROOM-1-N8R8
- RF: RFM95W 915 MHz LoRa
- Power: Dual AMS1117 series (VBAT → 5V → 3.3V)
- Pyro: 2-channel MOSFET deployment
- Sensors: BMP180, MPU6050, SHT40
- GPS: Adafruit breakout
- Storage: MicroSD (SPI)
- Target apogee: 1,000 m — LASC Rocket Challenge
